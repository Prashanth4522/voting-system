Key Features

Auto-Logout for Inactivity

Automatically logs users out after inactivity to prevent unauthorized access or voting misuse.

Voter Analytics

Admin dashboard shows total registered voters and total votes cast in real-time.

Facial Dataset Trainer

Admin can add/update facial data and re-train the model for better recognition across demographics.

Cross-Platform Support

Responsive design supports browsers and mobile devices with secure webcam access.

Encrypted Vote Storage

Votes are securely encrypted and optionally stored on blockchain for tamper-proof logging.

One-Time Secure Voting

Each user can vote only once after verifying their identity. Candidate details shown clearly.

Facial Login & Registration

Users register and log in using live facial capture and verification for high-security access.

Candidate Profiles

Voters can view candidate names, party logos, and short bios before making a voting decision.

Fraud Detection System

Alerts admin on multiple faces in frame, suspicious login attempts, or duplicate votes.
